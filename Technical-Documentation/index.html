Hello World

The Python language has many similarities to Perl, C, and Java. However, there are some definite differences between the languages.

To print your first 'Hello World' program. Type the following text at the Python prompt and press the Enter:

>>> print "Hello World"
Variable Types

Variables are nothing but reserved memory locations to store values. This means that when you create a variable you reserve some space in memory.

Based on the data type of a variable, the interpreter allocates memory and decides what can be stored in the reserved memory. Therefore, by assigning different data types to variables, you can store integers, decimals or characters in these variables.

Python has five standard data types:

  • Numbers
  • String
  • List
  • Tuple
  • Dictionary

To assign a value to a variable follow the next example:

counter =100 # An integer assignment
miles = 1000.0 # A floating point
name = "John" # A string
Basic Operators

Arithmetic Operator:

  • Addition: +
  • Substraction: -
  • Multiplication: *
  • Division: /
  • Modulus: %
  • Exponent: **
  • Floor division: //
a = 10
b = 5
a + b = 15

Comparison Operators:

  • Equal: ==
  • Not Equal: !=
  • Greater Than: >
  • Less Than: <
  • Greater or Equal: >=
  • Less or Equal: <=
a = 10
b = 10
a == b # TRUE
While Loop

Repeats a statement or group of statements while a given condition is TRUE. It tests the condition before executing the loop body. Example:

while (condition) statement
For Loop

Executes a sequence of statements multiple times and abbreviates the code that manages the loop variable.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
print(x)
